# Safari Support

This directory contains the Safari Web Extension wrapper for OverCite.

## Structure

- `OverCite.xcodeproj` — Xcode project
- `OverCite/` — macOS container app
- `OverCite Extension/` — bundled Safari extension

## Build Instructions

1. Open `OverCite.xcodeproj` in Xcode
2. Run the "OverCite" target
3. Enable the extension in Safari → Safari Extensions

## Notes

- The original extension source remains in `../extension/`
- The Safari bundle contains a copy with minimal modifications required for compatibility
- A lot of these changes were done using Codex and chatGPT. This README file was also generated by chatGPT.
