# OverCite v0.3.0

OverCite is now broader than ADS/SciX while keeping the fast ADS/SciX-only workflow as the default.

## Highlights

- Added source presets for astronomy, physics, math, computer science, life sciences, and cross-field work.
- Added arXiv, INSPIRE, Crossref, PubMed, and DataCite search routes.
- Added custom source routing with one primary database and optional fallbacks.
- Added `Author:Year` citation keys.
- Improved broad-source ranking for first authors, repeated records, arXiv preprints, datasets, and software.
- Renamed direct lookup wording to `Raw query`.
