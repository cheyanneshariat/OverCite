# OverCite v0.3.2

This release makes the TeXstudio beta broadly testable from the public repo while leaving the Chrome, Firefox, and VS Code v0.3.0 behavior intact.

## TeXstudio

- Added a local TeXstudio macro + Node CLI workflow using the v0.3.0 OverCite source-routing and insertion core.
- Added `./install-texstudio.sh` and `texstudio/scripts/install.mjs` to generate ready-to-import macros.
- Added `OverCite: Open Settings` so TeXstudio users can edit the JSON settings file from inside TeXstudio.
- Added `--doctor` setup diagnostics for Node, generated macros, settings files, and TeXstudio app detection on macOS.
- Added token-free quick-check files in `texstudio/fixtures/quick-check`.
- Hardened macro error handling so stale response files do not mask current failures.

## Validation

- TeXstudio automated suite: check, unit tests, smoke test, and doctor.
- Browser extension suite and build.
- VS Code extension check and tests.
- Manual macOS TeXstudio pass including first-run permissions, focus switching away and back, raw DOI lookup, editor-buffer update, and `.bib` write.
