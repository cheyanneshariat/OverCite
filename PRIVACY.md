# Privacy Policy

OverCite is a client-side extension. It does not use an OverCite backend, does not collect analytics, and does not store project text on an OverCite server.

OverCite stores its settings in browser storage, including optional API keys and user preferences such as source coverage, search mode, and citation-key settings.

When you use OverCite in Overleaf, it reads the current citation placeholder and nearby manuscript text from the active Overleaf page so it can search for likely matches. It sends search queries to the literature databases you selected in settings, such as ADS/SciX, arXiv, INSPIRE, Crossref, PubMed, or DataCite. When you choose a result, OverCite may send the selected record id, DOI, arXiv id, or PubMed id to the relevant database to retrieve or build BibTeX.

OverCite does not sell user data, does not use user data for advertising, and does not use user data for any purpose unrelated to the extension's citation-insertion workflow.
