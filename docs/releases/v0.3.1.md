# OverCite v0.3.1

Small bug-fix release for raw query mode.

## Fixed

- ADS/SciX raw query now preserves comma-separated fielded queries such as `author:"Muller, S." author:"Beelen, A." aff:"LAM", year:2026`.
- Fielded ADS/SciX raw queries now keep ADS/SciX result order instead of applying extra OverCite ranking.
- ADS/SciX raw query requests a larger result window than contextual search.

## Checks

- Browser extension tests: `145/145`
- VS Code extension tests: `128/128`
- Live raw-query checks: ADS/SciX, Crossref, arXiv, INSPIRE, and DataCite
