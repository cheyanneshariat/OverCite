# OverCite v0.3.0

Short version: broader search, same simple workflow.

## Highlights

- Source presets now cover more research areas.
- New sources: arXiv, INSPIRE, Crossref, PubMed, and DataCite.
- ADS/SciX-only stays the default fast path.
- Custom routing lets you choose one primary database and optional backups.
- Citation keys now support `Author:Year`.
- Ranking is better for first authors, duplicate records, arXiv preprints, datasets, and software.
- Direct lookup is now called `Raw query`.

## Update Notes

- Existing settings are preserved, including saved ADS/SciX tokens.
- ADS/SciX-only remains the default fast path for existing astronomy/astrophysics users.
- Browser users may see a one-time permission prompt only if they choose one of the new non-ADS databases.
