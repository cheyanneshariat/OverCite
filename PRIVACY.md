# Privacy Policy

OverCite is a client-side extension. It does not use an OverCite backend, does not collect analytics, and does not store project text on an OverCite server.

OverCite stores its settings in browser storage, including the ADS/SciX API token and user preferences such as search mode and citation-key settings.

When you use OverCite in Overleaf, it reads the current citation placeholder and nearby manuscript text from the active Overleaf page so it can search for likely matches. It sends the search query and selected bibcode to ADS/SciX only to retrieve search results and export BibTeX for the paper you choose.

OverCite does not sell user data, does not use user data for advertising, and does not use user data for any purpose unrelated to the extension's citation-insertion workflow.
